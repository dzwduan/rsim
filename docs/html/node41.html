Bus parameters

buswidth
This number specifies the width of the bus, in bytes. Defaults to 32.

buscycle
The number listed here gives the bus cycle time in processor cycles. Defaults to 3.

busarbdelay
This parameter gives the bus arbitration delay in processor cycles. Defaults to 1.
