Overall system parameters

numnodes
The number given with this parameter specifies the number of nodes in the system. Defaults to 16.
reqsz
This parameter gives the length of the control header for each packet, in bytes. The control header includes the requested address, the source node, the destination node, and the command type for the packet. Defaults to 16.
